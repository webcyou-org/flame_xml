# Flame XML Parser

## xml parser using Flame game engine

## Usage

### import IsometricTileMapComponent Class

```dart
import 'package:flame_xml/flame_xml.dart';
```

```dart
const filePath = 'tiles/tile_map.tsx';
final flameXML = await FlameXML.create(filePath);
```
